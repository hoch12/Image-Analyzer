# Image Analyzer

Image Analyzer is a Python application that uses Machine Learning to detect if an image contains a "middle finger" gesture (`fuck you`). The project fulfills all requirements for the PV AI/ML course, including a real-world use case, documentation, a custom hand-made dataset of &gt;1500 images, and a model trained from scratch.

## Project Structure
- `data/` - Contains the dataset (collected by the author).
- `data_collection/` - Scripts used to collect our own dataset via Webcam or download.
- `docs/` - Comprehensive documentation of the project analysis, workflow, and architecture.
- `lib/` (or `vendor/`) - Third-party code.
- `src/` - Our own project source code.

## Virtual Environment Setup
Before running any script, you must set up the virtual environment:
1. Open up a terminal.
2. Run `python -m venv .venv` to create a virtual environment.
3. Activate it:
   - On Windows: `.venv\Scripts\activate`
   - On Mac/Linux: `source .venv/bin/activate`
4. Install requirements: `pip install -r requirements.txt` (This file will be created as we progress).
